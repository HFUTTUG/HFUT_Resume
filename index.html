反馈问题

如果发现模板有问题，请按照以下步骤操作：

  1. 阅读常见问题 FAQ;
  2. 在 GitHub Issues 中搜索该问题的关键词;
  3. 在 GitHub Issues中提出新 issue;

  4. 提交 Pull Request 请确保提交的版本能够通过 Tex Build Test.

  5. 你也可以加入HFUT_Thesis的QQ群（904943223）进行讨论交流。
